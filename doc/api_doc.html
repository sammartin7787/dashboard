Solfox Dashboard¶

This is API documentation for Solfox Dashboard.

APIs¶

Here, Django views are defined for processing a user's request and for returning the response. A response data is json format.

def login ( request ) ¶

This function checks authorizatin of a user based on a user name and a password that are requested by a user.

Request url¶

url: http://54.187.32.139/static/index.html#/login

Method: post

Request Data¶

{'username':'username', 'password':'password'}

Response Data¶

If a username and a password are correct, the function returns login success message with json format.

{"login": "success"}

If a username or a password is incorrect, the function returns login failed message with json format.

{"login": "failed"}

def upload ( request ) ¶

This function accepts ajax request with file name and data, and saves the file in server side.

Request url¶

url: http://54.187.32.139/static/index.html#/index/upload

Method: post

Request Data¶ 
{
	'file': 
		{
			'filename': 'filename',
			'filedata': 'binary data of file'
		}
}
Response Data¶

If uploading a file is succeeded, the function returns upload success message with json format.

{"upload": "success"}

If uploading a file is failed, the function returns upload failed message with json format.

{"upload": "failed"}
